GitHub 연결 설정 ("업데이트" 버튼용)

"업데이트"(전체 재수집)는 이 저장소에 대한 Contents: Read/Write, Actions: Read/Write 권한을 가진 fine-grained Personal Access Token(PAT)이 필요합니다. 토큰은 이 브라우저의 localStorage에만 저장되며 어디로도 전송되지 않습니다(GitHub API 호출 시에만 사용).

키워드 검색 설정 ("키워드 추가"용)

"키워드 추가"는 Cloudflare Worker를 통해 즉시 검색 결과를 보여줍니다. Worker URL과, Worker에 등록해둔 앱 시크릿(무단 호출 방지용)을 입력하세요. 이 값들도 브라우저에만 저장됩니다.
